## Windows

1. 下载 [Typora 1.9.4 安装文件](win64/typora-setup-x64-1.9.4.exe)，并记住安装路径。
2. 下载 [winmm.dll](win64/winmm.dll)，并将其替换到 Typora 安装路径下包含 `typora.exe` 的文件夹中。
3. 重启 Typora，即可激活。
